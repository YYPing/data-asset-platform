# 数据资产平台 - 项目需求摘要

## 项目概述
政府数据资产登记试点支撑平台，实现数据资产从登记到退出的全生命周期管理。

## 10阶段生命周期
登记 → 合规评估 → 权属确认 → 确权登记 → 价值评估 → 入账 → 运营 → 变更管理 → 监管 → 退出/注销

## 7个角色
1. holder_admin（持有方管理员）- 管理本单位资产和用户
2. holder_user（持有方操作员）- 登记资产、上传材料
3. center_admin（登记中心管理员）- 管理审批流程和系统
4. center_user（登记中心操作员）- 审核资产、处理审批
5. assessor（第三方评估机构）- 提交评估报告
6. auditor（审计人员）- 只读+查看审计日志
7. sys_admin（系统管理员）- 系统配置、用户管理

## 流程状态机（12状态）
draft → pre_review → submitted → reviewing → correction → approved → publicizing → effective → changing → archived → cancelled → rejected

## 审批规则
- 普通资产：2级（持有方→登记中心初审→终审）
- 重大资产(>100万)：3级（+第三方评估）
- 超时：预审1天、初审5天、第三方15天、终审3天、补正10天(超30天自动关闭)、公示5天

## 材料管理
- 各阶段标准化材料清单，SHA256哈希存证
- 格式白名单：pdf/jpg/png/xlsx/docx
- MinIO存储，版本控制

## 登记证书
- 支持外部导入登记证书（PDF/图片）
- 记录证书编号、颁发机构、有效期
- 到期前30/15/7天自动提醒

## 大屏展示
- 持有方大屏：资产总览、审批进度、材料完整度
- 登记中心大屏：全市统计、完成率、审批效率、单位排名

## 技术约束
- 并发≥200，核心API P95<200ms
- 等保三级，JWT+RBAC
- 内网部署，Docker Compose
