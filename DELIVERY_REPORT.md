## 部署指南

### 快速启动
```bash
# 1. 克隆项目
git clone https://github.com/YYPing/data-asset-platform.git

# 2. 进入项目目录
cd data-asset-platform

# 3. 启动开发环境
./scripts/start-local.sh

# 4. 访问应用
# 前端: http://localhost:3000
# 后端: http://localhost:8000
# API文档: http://localhost:8000/docs
```

### 生产部署
```bash
# 1. 配置环境变量
cp .env.example .env
# 编辑 .env 文件配置数据库、Redis、MinIO等

# 2. 启动Docker服务
docker-compose up -d

# 3. 初始化数据库
docker-compose exec backend alembic upgrade head

# 4. 访问应用
# 前端: http://your-domain
# 后端API: http://your-domain/api
```
