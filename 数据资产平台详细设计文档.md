### 1.2 技术栈选型

#### 前端技术栈
- 核心框架: Vue.js 3.x + TypeScript
- UI组件库: Element Plus
- 状态管理: Pinia
- 路由管理: Vue Router 4.x
- HTTP客户端: Axios
- 图表库: ECharts 5.x
- 大屏框架: DataV
- 地图组件: 高德地图 API
- 移动端: Uni-app
- 构建工具: Vite

#### 后端技术栈
- 核心框架: Python 3.10+ FastAPI
- ORM框架: SQLAlchemy 2.0
- 数据验证: Pydantic
- 异步任务: Celery + Redis
- API文档: OpenAPI (Swagger)
- 认证授权: JWT + OAuth2
- 日志框架: Loguru
- 测试框架: Pytest

#### 数据存储
- 关系数据库: PostgreSQL 14+
- 对象存储: MinIO
- 缓存: Redis 7.x
- 全文搜索: Elasticsearch 8.x
- 消息队列: RabbitMQ

#### 部署运维
- 容器化: Docker + Docker Compose
- CI/CD: GitLab CI / GitHub Actions
- 监控: Prometheus + Grafana
- 日志: ELK Stack
- 反向代理: Nginx

## 三、API接口设计

### 3.1 认证授权接口
- POST /api/v1/auth/login - 用户登录
- POST /api/v1/auth/refresh - 刷新Token
- POST /api/v1/auth/logout - 用户登出

### 3.2 数据资产接口
- POST /api/v1/assets - 创建数据资产
- GET /api/v1/assets - 查询数据资产列表
- GET /api/v1/assets/{id} - 获取数据资产详情
- PUT /api/v1/assets/{id} - 更新数据资产
- DELETE /api/v1/assets/{id} - 删除数据资产

### 3.3 材料管理接口
- POST /api/v1/materials/upload - 上传材料
- GET /api/v1/materials - 查询材料列表
- GET /api/v1/materials/{id} - 获取材料详情
- PUT /api/v1/materials/{id} - 更新材料信息
- DELETE /api/v1/materials/{id} - 删除材料

### 3.4 大屏展示接口
- GET /api/v1/dashboard/holder/stats - 数据持有方统计
- GET /api/v1/dashboard/center/stats - 登记中心全市统计
