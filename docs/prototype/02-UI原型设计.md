# 数据资产全流程管理平台 - UI原型设计

## 1. 设计规范

### 1.1 色彩规范
```css
/* 主色调 */
--primary-color: #1890FF;        /* 主品牌色 */
--primary-light: #40A9FF;        /* 主色浅色 */
--primary-dark: #096DD9;         /* 主色深色 */

/* 辅助色 */
--success-color: #52C41A;        /* 成功 */
--warning-color: #FAAD14;        /* 警告 */
--error-color: #F5222D;          /* 错误 */

/* 中性色 */
--text-primary: #262626;         /* 主文字 */
--text-secondary: #595959;       /* 次要文字 */
--text-disabled: #BFBFBF;        /* 禁用文字 */
--border-color: #D9D9D9;         /* 边框 */
--background-color: #F5F5F5;     /* 背景 */
```

### 1.2 布局规范
- **侧边栏宽度**: 208px
- **顶部导航高度**: 64px
- **内容区边距**: 24px
- **卡片圆角**: 8px
- **字体**: 系统默认字体栈

## 4. 响应式设计说明

### 4.1 断点定义
- **大屏 (≥1920px)**: 大屏展示模式，完整布局
- **桌面 (≥1440px)**: 标准桌面布局
- **笔记本 (≥1280px)**: 笔记本适配
- **平板 (≥768px)**: 平板布局，侧边栏可收起

### 4.2 适配策略
- 管理后台: 主要支持桌面端，最小宽度1280px
- 客户门户: 支持响应式，适配平板和桌面
- 大屏展示: 固定1920×1080分辨率，全屏显示
