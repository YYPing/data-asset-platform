# 模块设计 - 系统登记与价值评估模块

## 1. 模块概述

### 1.1 功能定位
系统登记模块是数据资产梳理的起点，负责记录客户的IT系统信息；价值评估模块基于登记信息对系统进行多维度评估，筛选出高价值的数据资产进行深度挖掘。

### 1.2 业务流程
```
系统登记 → 信息完善 → 自动评估 → 人工复核 → 评估报告 → 系统筛选
```

## 3. 价值评估功能设计

### 3.1 评估模型设计

#### 3.1.1 评估维度
```java
@Component
public class SystemAssessmentEngine {
    
    /**
     * 执行系统价值评估
     */
    public AssessmentResult assess(CustomerSystem system) {
        AssessmentResult result = new AssessmentResult();
        
        // 1. 业务价值评估 (40%)
        double businessValue = assessBusinessValue(system);
        result.setBusinessValueScore(businessValue);
        
        // 2. 数据质量评估 (30%)
        double dataQuality = assessDataQuality(system);
        result.setDataQualityScore(dataQuality);
        
        // 3. 合规风险评估 (20%)
        double complianceRisk = assessComplianceRisk(system);
        result.setComplianceRiskScore(complianceRisk);
        
        // 4. 技术可行性评估 (10%)
        double technicalFeasibility = assessTechnicalFeasibility(system);
        result.setTechnicalFeasibilityScore(technicalFeasibility);
        
        // 计算综合评分
        double totalScore = businessValue * 0.4 + 
                           dataQuality * 0.3 + 
                           (100 - complianceRisk) * 0.2 + 
                           technicalFeasibility * 0.1;
        result.setTotalScore(totalScore);
        
        // 生成建议
        generateRecommendations(result);
        
        return result;
    }
    
    /**
     * 业务价值评估
     */
    private double assessBusinessValue(CustomerSystem system) {
        double score = 0;
        
        // 业务关键性 (30%)
        int criticality = system.getBusinessCriticality();
        score += normalize(criticality, 1, 5) * 30;
        
        // 用户覆盖度 (25%)
        int userCount = system.getUserCount();
        if (userCount >= 1000) score += 25;
        else if (userCount >= 500) score += 20;
        else if (userCount >= 100) score += 15;
        else if (userCount >= 50) score += 10;
        else score += 5;
        
        // 交易量重要性 (25%)
        int dailyVolume = system.getDailyTransactionVolume();
        if (dailyVolume >= 100000) score += 25;
        else if (dailyVolume >= 50000) score += 20;
        else if (dailyVolume >= 10000) score += 15;
        else if (dailyVolume >= 1000) score += 10;
        else score += 5;
        
        // 战略对齐度 (20%)
        String domain = system.getBusinessDomain();
        Set<String> strategicDomains = Set.of("销售", "财务", "生产");
        if (strategicDomains.contains(domain)) {
            score += 20;
        } else {
            score += 12;
        }
        
        return score;
    }
    
    /**
     * 数据质量评估
     */
    private double assessDataQuality(CustomerSystem system) {
        double score = 0;
        
        // 开发商声誉 (40%)
        String vendor = system.getVendor();
        score += getVendorReputationScore(vendor) * 0.4;
        
        // 系统成熟度 (30%)
        score += assessSystemMaturity(system) * 0.3;
        
        // 数据治理水平 (30%)
        score += assessDataGovernance(system) * 0.3;
        
        return score;
    }
    
    /**
     * 合规风险评估
     */
    private double assessComplianceRisk(CustomerSystem system) {
        double riskScore = 0;
        
        // 等保级别风险
        String securityLevel = system.getSecurityLevel();
        switch (securityLevel) {
            case "一级": riskScore += 40; break;
            case "二级": riskScore += 25; break;
            case "三级": riskScore += 15; break;
            case "四级": riskScore += 10; break;
            default: riskScore += 50;
        }
        
        // 数据分级风险
        String dataClassification = system.getDataClassification();
        switch (dataClassification) {
            case "机密": riskScore += 30; break;
            case "敏感": riskScore += 20; break;
            case "内部": riskScore += 10; break;
            case "公开": riskScore += 5; break;
            default: riskScore += 25;
        }
        
        // 历史合规记录
        if (system.getLastSecurityAssessment() != null) {
            riskScore -= 10; // 有评估记录降低风险
        }
        
        return Math.min(riskScore, 100);
    }
    
    /**
     * 技术可行性评估
     */
    private double assessTechnicalFeasibility(CustomerSystem system) {
        double score = 0;
        
        // 数据库类型可接入性 (60%)
        String dbType = system.getDatabaseType();
        Set<String> easyAccessDbs = Set.of("MySQL", "PostgreSQL", "Oracle", "SQL Server");
        if (easyAccessDbs.contains(dbType)) {
            score += 60;
        } else {
            score += 40;
        }
        
        // 数据量可行性 (40%)
        String volume = system.getEstimatedDataVolume();
        if (volume.contains("TB")) {
            if (volume.contains("100")) score += 20; // >100TB
            else score += 30; // 10-100TB
        } else {
            score += 40; // <10TB
        }
        
        return score;
    }
    
    /**
     * 生成评估建议
     */
    private void generateRecommendations(AssessmentResult result) {
        double totalScore = result.getTotalScore();
        
        if (totalScore >= 80) {
            result.setRecommendation("immediate");
            result.setRecommendationText("立即挖掘");
            result.setPriorityLevel("high");
            result.setEstimatedRoi(0.35);
        } else if (totalScore >= 60) {
            result.setRecommendation("priority");
            result.setRecommendationText("优先挖掘");
            result.setPriorityLevel("medium");
            result.setEstimatedRoi(0.25);
        } else if (totalScore >= 40) {
            result.setRecommendation("evaluate");
            result.setRecommendationText("评估后决定");
            result.setPriorityLevel("low");
            result.setEstimatedRoi(0.15);
        } else {
            result.setRecommendation("skip");
            result.setRecommendationText("暂不挖掘");
            result.setPriorityLevel("none");
            result.setEstimatedRoi(0.05);
        }
        
        // 建议的数据产品
        List<String> suggestedProducts = new ArrayList<>();
        if (result.getBusinessValueScore() > 80) {
            suggestedProducts.add("客户360视图");
        }
        if (result.getDataQualityScore() > 70) {
            suggestedProducts.add("业务分析报表");
        }
        if (result.getTotalScore() > 75) {
            suggestedProducts.add("决策支持系统");
        }
        result.setSuggestedProducts(suggestedProducts);
    }
}
```

## 5. 业务规则

### 5.1 自动评估触发条件
- 系统状态变为"registered"
- 系统关键信息（业务关键性、用户数量等）发生变化

### 5.2 评估审核流程
```
自动评估 → 评估师复核 → 提交审核 → 主管审批 → 评估完成
```

### 5.3 系统筛选规则
- 综合评分 >= 80分：自动标记为高优先级
- 综合评分 60-80分：人工决定是否挖掘
- 综合评分 < 60分：暂不挖掘，记录原因
