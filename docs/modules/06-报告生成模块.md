# 模块设计 - 报告生成与交付模块

## 1. 模块概述

### 1.1 功能定位
报告生成模块负责将项目各阶段的数据和分析结果整合为专业的报告文档，支持多种格式输出（PDF、Word、在线查看），并提供客户门户的文档交付功能。

### 1.2 报告类型
1. **系统登记报告**: 记录客户IT系统梳理结果
2. **价值评估报告**: 系统价值评估分析
3. **数据产品说明书**: 数据产品设计文档
4. **合规评估报告**: 法规符合性检查结果
5. **估值报告**: 数据资产估值结果
6. **综合交付报告**: 项目整体总结

## 4. 报告生成引擎

### 4.1 报告数据聚合
```java
@Service
public class ReportDataAggregator {
    
    /**
     * 聚合系统登记报告数据
     */
    public SystemRegistryReportData aggregateRegistryData(Long projectId) {
        SystemRegistryReportData data = new SystemRegistryReportData();
        
        // 项目信息
        Project project = projectRepository.findById(projectId).orElseThrow();
        data.setProject(project);
        
        // 客户信息
        Customer customer = customerRepository.findById(project.getCustomerId()).orElseThrow();
        data.setCustomer(customer);
        
        // 系统列表
        List<CustomerSystem> systems = systemRepository.findByProjectId(projectId);
        data.setSystems(systems);
        
        // 统计信息
        data.setTotalSystems(systems.size());
        data.setSystemTypeDistribution(systems.stream()
            .collect(Collectors.groupingBy(
                CustomerSystem::getSystemType, 
                Collectors.counting()
            )));
        data.setBusinessDomainDistribution(systems.stream()
            .collect(Collectors.groupingBy(
                CustomerSystem::getBusinessDomain,
                Collectors.counting()
            )));
        
        return data;
    }
    
    /**
     * 聚合价值评估报告数据
     */
    public ValuationReportData aggregateValuationData(Long projectId) {
        ValuationReportData data = new ValuationReportData();
        
        // 基础信息
        data.setProject(projectRepository.findById(projectId).orElseThrow());
        
        // 评估结果
        List<SystemAssessment> assessments = assessmentRepository
            .findByProjectId(projectId);
        data.setAssessments(assessments);
        
        // 统计分析
        double avgScore = assessments.stream()
            .mapToDouble(a -> a.getTotalScore().doubleValue())
            .average()
            .orElse(0);
        data.setAverageScore(BigDecimal.valueOf(avgScore));
        
        long highValueCount = assessments.stream()
            .filter(a -> a.getTotalScore().doubleValue() >= 80)
            .count();
        data.setHighValueSystemCount((int) highValueCount);
        
        // 建议数据产品汇总
        List<String> allSuggestedProducts = assessments.stream()
            .flatMap(a -> a.getSuggestedDataProducts().stream())
            .distinct()
            .collect(Collectors.toList());
        data.setSuggestedProducts(allSuggestedProducts);
        
        return data;
    }
    
    /**
     * 聚合合规评估报告数据
     */
    public ComplianceReportData aggregateComplianceData(Long assessmentId) {
        ComplianceReportData data = new ComplianceReportData();
        
        ComplianceAssessment assessment = complianceRepository
            .findById(assessmentId)
            .orElseThrow();
        data.setAssessment(assessment);
        
        // 检查项
        List<ComplianceCheckItem> checkItems = checkItemRepository
            .findByAssessmentId(assessmentId);
        data.setCheckItems(checkItems);
        
        // 按法规分组统计
        Map<String, List<ComplianceCheckItem>> groupedByRegulation = 
            checkItems.stream()
                .collect(Collectors.groupingBy(
                    ComplianceCheckItem::getRegulationType
                ));
        data.setGroupedCheckItems(groupedByRegulation);
        
        // 风险统计
        long highRiskCount = checkItems.stream()
            .filter(item -> "high".equals(item.getRiskLevel()))
            .count();
        data.setHighRiskItemCount((int) highRiskCount);
        
        // 律师意见
        List<ComplianceDocument> legalOpinions = documentRepository
            .findByAssessmentIdAndType(assessmentId, "legal_opinion");
        data.setLegalOpinions(legalOpinions);
        
        // 制度材料
        List<ComplianceDocument> policyDocs = documentRepository
            .findByAssessmentIdAndType(assessmentId, "policy_doc");
        data.setPolicyDocuments(policyDocs);
        
        return data;
    }
}
```

### 4.2 报告渲染引擎
```java
@Service
public class ReportRenderEngine {
    
    @Autowired
    private TemplateEngine templateEngine;
    
    /**
     * 渲染报告
     */
    public String renderReport(String templateCode, Object reportData) {
        // 加载模板
        ReportTemplate template = templateRepository
            .findByTemplateCode(templateCode);
        
        // 创建Thymeleaf上下文
        Context context = new Context();
        context.setVariable("data", reportData);
        context.setVariable("reportDate", LocalDate.now());
        context.setVariable("reportNumber", generateReportNumber());
        
        // 添加工具函数
        context.setVariable("formatMoney", new MoneyFormatter());
        context.setVariable("formatPercent", new PercentFormatter());
        context.setVariable("formatDate", new DateFormatter());
        
        // 渲染HTML
        String htmlContent = templateEngine.process(
            template.getTemplateContent(), 
            context
        );
        
        return htmlContent;
    }
    
    /**
     * 生成PDF
     */
    public byte[] generatePdf(String htmlContent) {
        try {
            ByteArrayOutputStream outputStream = new ByteArrayOutputStream();
            
            // 使用OpenPDF或Flying Saucer生成PDF
            ITextRenderer renderer = new ITextRenderer();
            
            // 添加中文字体支持
            renderer.getFontResolver().addFont(
                "classpath:fonts/SimSun.ttf", 
                BaseFont.IDENTITY_H, 
                BaseFont.NOT_EMBEDDED
            );
            
            renderer.setDocumentFromString(htmlContent);
            renderer.layout();
            renderer.createPDF(outputStream);
            
            return outputStream.toByteArray();
        } catch (Exception e) {
            throw new ReportGenerationException("PDF生成失败", e);
        }
    }
    
    /**
     * 生成Word文档
     */
    public byte[] generateWord(String htmlContent) {
        try {
            // 使用POI将HTML转换为Word
            XWPFDocument document = new XWPFDocument();
            
            // 解析HTML并转换为Word段落
            Document htmlDoc = Jsoup.parse(htmlContent);
            Elements paragraphs = htmlDoc.select("p, h1, h2, h3, h4, li");
            
            for (Element para : paragraphs) {
                XWPFParagraph wordPara = document.createParagraph();
                XWPFRun run = wordPara.createRun();
                run.setText(para.text());
                
                // 设置样式
                if (para.tagName().startsWith("h")) {
                    int level = Integer.parseInt(para.tagName().substring(1));
                    wordPara.setStyle("Heading" + level);
                }
            }
            
            ByteArrayOutputStream outputStream = new ByteArrayOutputStream();
            document.write(outputStream);
            
            return outputStream.toByteArray();
        } catch (Exception e) {
            throw new ReportGenerationException("Word生成失败", e);
        }
    }
}
```

## 5. API接口

### 5.1 生成报告
```yaml
接口: POST /api/reports/generate
请求体:
  projectId: long, required
  reportType: string, required
  templateId: long
响应:
  code: 200
  data:
    reportId: long
    reportName: string
    downloadUrl: string
```

### 5.2 获取报告
```yaml
接口: GET /api/reports/{id}
响应:
  code: 200
  data:
    reportId: long
    reportType: string
    reportName: string
    content: string  # HTML内容
    pdfUrl: string
    wordUrl: string
    status: string
```

### 5.3 下载报告
```yaml
接口: GET /api/reports/{id}/download
参数:
  format: string  # pdf/word
响应:
  Content-Type: application/pdf 或 application/vnd.openxmlformats-officedocument.wordprocessingml.document
```

### 5.4 发布到客户门户
```yaml
接口: POST /api/reports/{id}/publish
请求体:
  visibleToCustomer: boolean
  customerPermissions: object
响应:
  code: 200
```
