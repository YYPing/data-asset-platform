## 📱 设计图导出格式

### 支持的导出格式：
1. **Markdown (.md)** - 原始设计文件
2. **HTML (.html)** - 网页查看
3. **PDF (.pdf)** - 打印和分享
4. **PNG/JPEG** - 图片格式（需要额外工具）

### 导出为PDF：
```bash

pandoc docs/prototype/02-UI原型设计.md -o ui-prototype.pdf

# 1. 先转换为HTML

```

## 📞 技术支持

如有设计图查看问题，可通过以下方式解决：

1. **文件权限问题**:
   ```bash
   chmod +r docs/**/*.md
   ```

2. **编码问题**:
   ```bash
   # 检查文件编码
   file -I docs/prototype/02-UI原型设计.md
   
   # 转换为UTF-8
   iconv -f GBK -t UTF-8 docs/prototype/02-UI原型设计.md > ui-utf8.md
   ```

3. **文件损坏**:
   ```bash
   # 重新下载或从备份恢复
   git checkout docs/prototype/02-UI原型设计.md
   ```
